Mécanique

Force gravitationnelle : F = G(m₁m₂)/r²

Poids : P = mg

Énergie cinétique : Ec = ½mv²

Énergie potentielle : Ep = mgh

Travail : W = F·d

Puissance : P = W/t = F·v

Cinématique

Vitesse moyenne : v = Δx/Δt

Accélération : a = Δv/Δt

MRUV - Position : x = x₀ + v₀t + ½at²

MRUV - Vitesse : v = v₀ + at

Vitesse finale : v² = v₀² + 2aΔx

MCU - Accélération : a = v²/r

Thermodynamique

Loi des gaz parfaits : PV = nRT

Premier principe : ΔU = Q + W

Chaleur : Q = mcΔT

Entropie : ΔS = Q/T

Loi de Boyle-Mariotte : P₁V₁ = P₂V₂

Rendement thermique : η = 1 - T_f/T_c

Électricité

Loi d'Ohm : U = RI

Puissance électrique : P = UI = RI²

Force de Coulomb : F = k(q₁q₂)/r²

Champ électrique : E = F/q = k(Q/r²)

Capacité : C = Q/U

Résistances série : R_eq = R₁ + R₂ + R₃

Magnétisme

Force de Lorentz : F = qv×B

Loi de Biot-Savart : dB = (μ₀/4π)(Idl×r/r³)

Flux magnétique : Φ = B·S

Loi de Faraday : ε = -dΦ/dt

Inductance : L = Φ/I

Force entre fils : F/l = (μ₀/2π)(I₁I₂/d)

Optique

Loi de Snell-Descartes : n₁sin(θ₁) = n₂sin(θ₂)

Lentille mince : 1/f = 1/p + 1/p'

Grandissement : γ = -p'/p = h'/h

Diffraction : sinθ = λ/a

Interférences : δ = 2d·sin(θ)

Energie photon : E = hν = hc/λ

Ondes

Vitesse de phase : v = λf

Effet Doppler : f' = f(v ± v_s)/(v ∓ v_o)

Onde sonore : v = √(γRT/M)

Intensité sonore : I = P/S

Niveau sonore : L = 10log(I/I₀)

Onde stationnaire : f_n = nv/2L

Physique Nucléaire

Équivalence masse-énergie : E = mc²

Énergie de liaison : E_l = Δm·c²

Loi de décroissance : N(t) = N₀e^(-λt)

Demi-vie : t₁/₂ = ln(2)/λ

Activité : A = λN

Défaut de masse : Δm = Zm_p + Nm_n - M

Relativité

Dilatation du temps : Δt = Δt₀/√(1-v²/c²)

Contraction des longueurs : L = L₀√(1-v²/c²)

Masse relativiste : m = m₀/√(1-v²/c²)

Quantité de mouvement : p = mv/√(1-v²/c²)

Énergie totale : E = mc² = m₀c²/√(1-v²/c²)

Physique Quantique

Relation de de Broglie : λ = h/p

Principe d'incertitude : ΔxΔp ≥ ħ/2

Équation de Schrödinger : iħ∂Ψ/∂t = ĤΨ

Énergie quantique : E = hf

Niveaux d'énergie : E_n = -13.6eV/n²

Spin : S = ±ħ/2
